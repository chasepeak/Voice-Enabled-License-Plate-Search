The following are necessary system requirements in order to run the RecordSearch application:

1) Java SE 11 or greater must be installed [1].
  -  For information on setting the default JDK for use by this application [2].

2) AWS credentials need to be set up on the local machine to use the web services of this program. Instructions for doing so can be found here [3].
  - The access and secret key can be made available upon request.

3) Executing the program is done by typing the following command into the command-line of your terminal:
  - java -jar [file path to RecordSearch.jar]
  - Any errors occurring, such as system issues with delegating an audio line for capturing user input, will appear in the terminal following the above command.

[1] https://www.oracle.com/technetwork/java/javase/downloads/jdk11-downloads-5066655.html)
[2] https://docs.oracle.com/cd/E19182-01/820-7851/inst_cli_jdk_javahome_t/
